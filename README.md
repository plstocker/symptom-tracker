Symptom tracket is really an anything tracker that I decided to build partly as a learning experience and partly because I needed it.
Since this is meant for keeping track of your health as its happening, I followed two principles:
1. All recorded measurements should be stored on the users device and easily accessible.
2. Taking a measurement should be as low friction as possible.

I also wanted to make this tool as flexible as possible, so I made it customizable. All of the "generic" symptoms that are loaded by default can be changed or removed. 
New custom measurements can be added.
